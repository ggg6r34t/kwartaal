One shared shell: mark + wordmark over a single ~400px card on paper. Magic link is the hero path everywhere; password is a quiet disclosure. Anti-enumeration throughout — known and unknown addresses render identically. Errors reassure and route forward. Auth is a hallway, not a lobby.

Calm under failure — most of these screens are seen eleven days before a deadline.
